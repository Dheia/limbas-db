Info

For a list of developers and other contributors, see the Credits appendix in the manual.

Start

You can access to the following applications:
LIMBAS framework LIMBAS SOAP access
LIMBAS WSDL access LIMBAS WEBDAV access

Documentation

A detailed description of how to perform a new INSTALLATION is provided at limbas.org.
The UPDATE process is documented in limbas.org.
The default URL of your LIMBAS installation is: http://localhost/openlimbas/dependent/index.php.
The default Installation PATH of Limbas is: "/srv/www/htdocs/openlimbas".

Important

Please send bug (error) reports, questions and comments to the mailing list at: sourceforge.net/projects/limbas/.

Notice

If you are running limbas demo database and you want to start with your own project you can clean all and start with a new instance by reset your system under admin->tools->reset database

Default Passwords

As default admin-user for the first login in LIMBAS use:
  • user: admin
  • password: limbas
For root user in appliances use:
  • user: root
  • pass: linux

Appliances

Following pre-build appliances are available for LIMBAS: (please contact us: vertrieb@limbas.org)
  • USB stick / hard disk image
  • Live CD/DVD (.iso)
  • VMware / VirtualBox / KVM (.vmdk)
  • OVF virtual machine (.ovf)
  • Xen guest
  • Preload ISO (.iso)
